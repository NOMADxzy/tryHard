## 栈

### 1. 概念
栈本质是一个线性表，所以它既可以是顺序表也可以是链表，通常为顺序表。与寻常的线性表的不同之处是：栈只可以在端点存入与输出（端点：一般为线性表的尾部，也就是栈顶），不可以随意的
的中间插入输出、头部插入输出。因为这个要求，造就了栈“先进后出，后进先出”的特点。    

`单调栈: ` 从栈底元素到栈顶元素呈单调递增或单调递减，栈内序列满足单调性的栈。
